# Website Rules

## Public Copy

- Keep positioning specific, measurable, and tied to business outcomes.
- Avoid generic "AI agency" claims when a narrower workflow/system offer is intended.
- Do not invent case studies, metrics, logos, or client outcomes.

## Static Files

- Preserve simple static-site structure unless the task explicitly asks for a framework.
- Keep links, forms, and contact CTAs aligned with the current offer and acquisition motion.

## Verification

- Inspect changed HTML/CSS manually.
- If visual behavior changed, capture screenshot evidence and reference the file path in the task summary.
